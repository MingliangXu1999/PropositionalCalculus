## 使用说明

具体使用方法如下：

```python
python calculus.py
```

根据命令提示不妨输入一个公式：

```
Input the T|-p:such as "|-p->~p"

p|-(p->p)->(p->p)
```

得到结果：

```
[[40, ['(p->p)->((p->p)->(p->p))', 'L1']], [6482, ['p->p', 1, 2, 'MP']], [6501, ['(p->p)->(p->p)', 6482, 40, 'MP']], [2, ['p->(p->p)', 'L1']]]
```
列表中元素与基本证明方法一致，此例中(1)为假定，即：
```
(1) p 假定
(2) p->(p->p) L1
(6482)  p->p (1),(2),MP
(40) (p->p)->((p->p)->(p->p)) L1
(6501) (p->p)->(p->p) (6482),(40),MP
```
